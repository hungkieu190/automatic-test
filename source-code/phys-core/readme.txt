== Description ==

Phys Core is a Core Processor of all **WordPress themes by PhysCode** and it is meant to be used only with PhysCode's WordPress themes.

# What does Phys Core do:
- Register necessary custom post types, custom fields for the themes
- Add a Control Panel, Customization Page to easier manage the theme
- Works as a One-Click Installer to install WordPress themes easier

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/phys-core` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Read our guide on how to Get Started with Phys Core.
